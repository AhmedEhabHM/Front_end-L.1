HTML Text Formatting

Bold text: This text is bold but lacks semantic meaning.

Strong text: This text is bold and indicates importance.

Italic text: This text is italic without semantic emphasis.

Emphasized text: This text is italic with semantic emphasis.

Highlighted text: Use this to draw attention to specific text.

Small text: Use this for less important content.

Deleted text: Strikethrough text can indicate revisions.

Inserted text: Underlined text can indicate new additions.

Superscript text: Example: x2 for exponents.

Subscript text: Example: H2O for chemical formulas.
